# MealPilot Domain Model

Family is the tenant and aggregate root.

Core entities: Family, Member, Goal, Meal, MealPlan, Recipe, Recommendation, Activity, Measurement, ShoppingList, Notification.

Roles: Owner, Adult, Child, Professional (future).

Domain events include MealRecommended, MealConsumed, ActivityRecorded, WeightRecorded and ShoppingListGenerated.
